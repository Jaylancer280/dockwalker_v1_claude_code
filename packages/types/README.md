# @dockwalker/types

Shared TypeScript type definitions for the DockWalker monorepo. This package has no runtime dependencies — it exports only types.

## Files

| File            | Contents                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| --------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `src/enums.ts`  | `RoleContext`, `IdentityType`, `ApplicationStatus` (includes `shortlisted`), `VesselType` (motor\|sail), `VesselOperation` (private\|charter), `ContractType` (permanent\|rotational\|seasonal\|crossing\|delivery\|temporary), `MealOption`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| `src/events.ts` | `EventType` (all domain event names including `EXPERIENCE.ADDED/UPDATED/REMOVED`, `ENGAGEMENT.POSTPONEMENT_*`, `ENGAGEMENT.WORK_STARTED`, `ENGAGEMENT.WORK_STARTED_CONFIRMED`, `DAYWORK.RELISTED`, `ENGAGEMENT.CANCELLATION_RATED_BY_*`, `CHECKLIST.SET`, `CHECKLIST.ITEM_TOGGLED`), `AggregateType` (includes `engagement`, `checklist`, `experience`), `DomainEvent` interface, `EventPayloadMap` (typed payload per event for compile-time validation). `EXPERIENCE.ADDED` carries full experience fields (vessel_operation, contract_type, contract_details) including salary (DB-only). `EXPERIENCE.UPDATED` carries all fields optional. `EXPERIENCE.REMOVED` is empty. `PROFILE.CREATED/UPDATED` payloads extended with green crew fields (`shore_experience`, `motivation`, `languages`, `available_to_start`, `onboarding_version`). `VESSEL.CREATED` carries `loa_meters: number` and `vessel_operation: string`; `VESSEL.UPDATED` carries optional `loa_meters` and `vessel_operation`. `CHECKLIST.SET` carries `engagement_id` and `items` array. `CHECKLIST.ITEM_TOGGLED` carries `engagement_id`, `item_id`, and `checked` boolean. `AVAILABILITY.SET` carries `start_date`, `end_date`, `expires_at`, `city_id`, optional `port_id`, and optional `not_available` boolean. |
| `src/models.ts` | Entity interfaces: `Person`, `CrewProfile` (extended with `shore_experience`, `motivation`, `languages`, `available_to_start`, `onboarding_version`), `AgentProfile`, `Vessel`, `Daywork`, `Application`, `AvailabilityWindow`, `Message`, `Engagement`, `EngagementRating`, `CrewExperience` (vessel_operation, contract_type, contract_details — no salary fields, DB intelligence only)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| `src/index.ts`  | Barrel re-export of all the above                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |

## Usage

```ts
import type { DomainEvent, EventType, EventPayloadMap, RoleContext } from '@dockwalker/types';
```

## Scripts

```bash
npm run type-check   # tsc --noEmit
```
