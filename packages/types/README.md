# @dockwalker/types

Shared TypeScript type definitions for the DockWalker monorepo. This package has no runtime dependencies — it exports only types.

## Files

| File            | Contents                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| --------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `src/enums.ts`  | `RoleContext`, `IdentityType`, `ApplicationStatus` (includes `shortlisted`), `VesselType` (motor\|sail), `VesselOperation` (private\|charter), `ContractType` (permanent\|rotational\|seasonal\|crossing\|delivery\|temporary), `MealOption`, `SubscriptionPlan` (free\|crew_pro\|crew_unlimited), `SubscriptionStatus` (active\|past_due\|cancelled\|trialing)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| `src/events.ts` | `EventType` (all domain event names including `EXPERIENCE.ADDED/UPDATED/REMOVED`, `ENGAGEMENT.POSTPONEMENT_*`, `ENGAGEMENT.WORK_STARTED`, `ENGAGEMENT.WORK_STARTED_CONFIRMED`, `DAYWORK.RELISTED`, `ENGAGEMENT.CANCELLATION_RATED_BY_*`, `CHECKLIST.SET`, `CHECKLIST.ITEM_TOGGLED`, `DAYWORK.INVITED`, `DAYWORK.INVITATION_ACCEPTED`, `DAYWORK.INVITATION_DECLINED`), `AggregateType` (includes `engagement`, `checklist`, `experience`, `invitation`, `admin`), `DomainEvent` interface, `EventPayloadMap` (typed payload per event for compile-time validation), `DayworkInvitation` model interface. `DAYWORK.INVITED` carries `daywork_id` + `crew_person_id`. `DAYWORK.INVITATION_ACCEPTED/DECLINED` carry `daywork_id` + `invitation_id`. `EXPERIENCE.ADDED` carries full experience fields (vessel_operation, contract_type, contract_details) including salary (DB-only). `EXPERIENCE.UPDATED` carries all fields optional. `EXPERIENCE.REMOVED` is empty. `PROFILE.CREATED/UPDATED` payloads extended with green crew fields (`shore_experience`, `motivation`, `languages`, `available_to_start`, `onboarding_version`). `VESSEL.CREATED` carries `loa_meters: number` (vessel_operation removed in Stage 69); `VESSEL.UPDATED` carries optional `loa_meters`. `DAYWORK.POSTED` carries optional `working_day_dates: string[]` and optional `positions_available: number`. `DAYWORK.POSITIONS_UPDATED` carries `daywork_id` + `positions_available`. `DAYWORK.EXTENDED` carries `daywork_id`, `end_date`, optional `working_days` and `working_day_dates`. `CHECKLIST.SET` carries `engagement_id` and `items` array. `CHECKLIST.ITEM_TOGGLED` carries `engagement_id`, `item_id`, and `checked` boolean. `AVAILABILITY.SET` carries `start_date`, `end_date`, `expires_at`, `city_id`, optional `port_id`, and optional `not_available` boolean. |
| `src/models.ts` | Entity interfaces: `Person`, `CrewProfile` (extended with `shore_experience`, `motivation`, `languages`, `available_to_start`, `onboarding_version`, `avatar_url`), `AgentProfile` (extended with `avatar_url`), `Vessel`, `Daywork` (extended with `positions_available`, `positions_filled`, `permanent_opportunity`), `Application`, `AvailabilityWindow`, `Message`, `Engagement`, `EngagementRating` (extended with `permanent_opportunity_accuracy`), `CrewExperience` (vessel_operation, contract_type, contract_details — no salary fields, DB intelligence only), `Subscription` (Stripe-owned state: plan, status, period)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| `src/index.ts`  | Barrel re-export of all the above                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |

## Usage

```ts
import type { DomainEvent, EventType, EventPayloadMap, RoleContext } from '@dockwalker/types';
```

## Scripts

```bash
npm run type-check   # tsc --noEmit
```
